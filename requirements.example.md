# Requirements

## Scale
- Single tenant
- <100 concurrent users
- <10k total records

## Priorities
- Working MVP first
- Simple > clever
- Ship in days, not weeks

## Skip
- Authentication (add later)
- Complex caching
- Microservices
- CI/CD pipelines
- Monitoring/alerting

## Quality Bar
- TypeScript strict mode
- Python type hints
- Tests for API endpoints
- No obvious security holes
